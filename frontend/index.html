Đang tải Đường đua xanh...

Trong trường hợp trình duyệt của bạn cũ đến mức không hỗ trợ công nghệ WebAssembly thì có lẽ bạn không phù hợp cho cuộc thi này.

Bạn có thể:
  • Xóa Internet Explorer nếu bạn đang dùng nó (wtf)
  • Sử dụng trình duyệt phiên bản cao lên. Nếu dùng Windows XP thì tải Supermium, Chromium mới nhất tương thích với Windows XP+
  • Bật JavaScript lên để nó load WebAssembly.
